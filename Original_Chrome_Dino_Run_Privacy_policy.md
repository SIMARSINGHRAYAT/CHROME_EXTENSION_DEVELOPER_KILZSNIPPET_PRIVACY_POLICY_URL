# Privacy Policy for Original Chrome Dino Run

**Last Updated:** April 23, 2026

## Overview
Original Chrome Dino Run is a browser extension designed to allow users to play the classic Chrome Dinosaur game at any time without requiring an internet disconnection. This extension is built with a strong commitment to user privacy and does not collect or process any personal data.

## Data Collection
This extension does not collect, store, or transmit any personally identifiable information or sensitive user data. This includes, but is not limited to, names, email addresses, financial information, authentication credentials, or personal communications.

## Usage of Data
Since no user data is collected, no data is used, processed, shared, or sold. The extension operates entirely within the user’s browser and functions independently without any backend servers or external data processing systems.

## Permissions
The extension only uses the minimum permissions required to function properly. It does not access browsing history, monitor user activity, or interact with website content beyond what is necessary to run the game.

## Cookies and Tracking
This extension does not use cookies, tracking technologies, analytics tools, or any form of user behavior monitoring.

## Third-Party Services
Original Chrome Dino Run does not use any third-party services, APIs, or external scripts that collect or process user data.

## Remote Code
All code required for the functionality of this extension is packaged within the extension itself. No remote code is executed, and no external scripts are loaded at runtime.

## Data Security
Since no user data is collected or stored, there is no risk of data breaches or unauthorized access to personal information through this extension.

## Changes to This Privacy Policy
This Privacy Policy may be updated from time to time to reflect changes in functionality or compliance requirements. Any updates will be reflected by modifying the "Last Updated" date at the top of this document.

## Contact Information
If you have any questions or concerns regarding this Privacy Policy, you may contact the developer through the support information provided in the Chrome Web Store listing.
